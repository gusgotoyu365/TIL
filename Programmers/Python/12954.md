풀이
-------------
n개가 x씩 증가.<br>
그걸 리스트에 넣어줬다.

```python
def solution(x, n):
    numList = []
    for i in range(1,n+1):
        numList.append(i * x)
    answer = numList
    return answer
```

처음에 틀렸던거
-------------
for 문이 2번 쓰이는 부분이 좀 거시기하고<br>
3번째 파라미터에 0이 들어오는 경우 런타임 에러가 났다.

```python
def solution(x, n):
    numList = []
    if (x > 0):
        for i in range(x,x*n+1,x):
            numList.append(i)
    else:
        for i in range(x,x*n-1,x):
            numList.append(i)
    answer = numList
    return answer
```
