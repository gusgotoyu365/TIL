풀고나서 다른 풀이를 찾아보니 완전제곱수 라는게 있댄다.
그게 약수가 홀수라는데
세상에 이런일이

```
def solution(left, right):
    answer = 0
    divisorNum = 0
    for i in range(left,right+1):
        for j in range(1,i+1):
            if(i%j==0):
                divisorNum += 1
                print(i,j,divisorNum)
        if (divisorNum % 2 == 0):
            answer += i
        else:
            answer -= i
        divisorNum = 0
    return answer
```
